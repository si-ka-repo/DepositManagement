## 📐 レイアウト仕様

### 構造
1. **ユニット名**: 
   - 太字、12pt
   - 上マージン: 8pt
   - 下マージン: 8pt

2. **セル区切りのテーブル**:
   - 外側ボーダー: 1px solid #000
   - セル間のボーダー: 1px solid #ccc
   - 1行目（利用者名）:
     - 背景色: #f0f0f0（薄いグレー）
     - 横並びでセルに分割
     - フォントサイズ: 10pt
     - パディング: 4pt
     - 各セル: 等幅または内容に応じた幅
   - 2行目（当月残高）:
     - 背景色: 白（デフォルト）
     - 利用者名の下に配置
     - 横並びでセルに分割
     - フォントサイズ: 10pt
     - 右寄せ
     - パディング: 4pt

3. **ユニット合計**:
   - テーブルの下、右寄せ
   - フォントサイズ: 11pt、太字
   - 上マージン: 8pt

### スタイル詳細

```typescript
unitBox: {
  border: "1px solid #000",
  padding: 8,
  marginBottom: 8,
}

unitName: {
  fontSize: 12,
  fontWeight: "bold",
  fontFamily: "NotoSansJP",
  marginBottom: 8,
}

tableContainer: {
  border: "1px solid #000",
  marginBottom: 8,
}

nameRow: {
  flexDirection: "row",
  flexWrap: "wrap",
  backgroundColor: "#f0f0f0",  // 薄いグレー
  borderBottom: "1px solid #ccc",
}

  flexDirection: "row",
  flexWrap: "wrap",
  borderBottom: "1px solid #000",  // 最後の行は太いボーダー

  padding: 4,

  flex: 1,  // 等幅に調整
}

  fontSize: 10,
  fontFamily: "NotoSansJP",
  textAlign: "left",
}

  fontSize: 10,
```

## 🎯 実装イメージ

### レイアウト構造

```
View (unitBox)
├── Text (unitName) - "ユニットA"
├── View (tableContainer)
│   ├── View (nameRow) - 背景 #f0f0f0
│   │   ├── View (cell + nameCell) - "山田太郎"
│   │   ├── View (cell + nameCell) - "鈴木一郎"
│   │   └── ...
│   └── View (valueRow)
│       ├── View (cell + valueCell) - "¥5,000"
│       ├── View (cell + valueCell) - "¥15,000"
│       └── ...
└── View (unitTotalRow)
    ├── Text (unitTotalLabel) - "ユニット合計:"
    └── Text (unitTotalValue) - "¥20,000"
```

## 📝 実装時の考慮事項

1. **セルの幅**:
   - 等幅にするか、内容に応じた幅にするか
   - `flex: 1`で等幅、`width`で固定幅

2. **利用者が多い場合**: 
   - `flexWrap: "wrap"`で自動的に折り返し
   - 折り返し時もセル構造を維持

3. **ボーダー**:
   - 外側: 1px solid #000（太い）
   - セル間: 1px solid #ccc（細い）
   - 最後の行の下: 1px solid #000（太い）

4. **背景色**:
   - 利用者名行: #f0f0f0（薄いグレー）
   - 残高行: 白（デフォルト）

5. **セルの配置**:
   - 利用者名: 左寄せ
   - 残高: 右寄せ

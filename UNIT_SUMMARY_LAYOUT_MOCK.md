## 📐 レイアウト仕様（パターン3をベース）

### 構造
1. **ユニット名**: 太字、12pt、上マージン8pt
2. **1行目（利用者名）**: 
   - 利用者名を横に並べる
   - フォントサイズ: 10pt
   - 利用者間の間隔: 20pt
   - 左マージン: 10pt
   - 上マージン: 8pt
3. **2行目（当月残高）**:
   - 各利用者の残高を横に並べる
   - フォントサイズ: 10pt
   - 利用者名の下に配置
   - 右寄せ
   - 上マージン: 4pt
4. **ユニット合計**:
   - 右下に配置
   - フォントサイズ: 11pt、太字
   - 上マージン: 8pt

## 🎯 推奨案

**パターン3**を推奨します。理由：
- シンプルで見やすい
- 利用者名と残高の対応が明確
- スペースを効率的に使用
- 実装が比較的簡単

## 📝 実装時の考慮事項

1. **利用者が多い場合**: 
   - `flexWrap: "wrap"`で自動的に折り返し
   - または固定幅で複数行に分割

2. **利用者名の長さ**:
   - 利用者名が長い場合は、残高の位置を調整
   - または利用者名を省略表示

3. **ユニット合計の位置**:
   - 右下に固定配置
   - 利用者の数に関わらず同じ位置

4. **レスポンシブ対応**:
   - PDFなので固定レイアウト
   - ただし、利用者が多い場合は折り返しを考慮

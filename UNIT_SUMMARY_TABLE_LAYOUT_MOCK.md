## 📐 レイアウト仕様

### 構造
1. **ユニット名**: 
   - 太字、12pt
   - 上マージン: 8pt
   - 下マージン: 8pt

2. **テーブル**:
   - ボーダー: 1px solid #000
   - ヘッダー行: 背景色 #f0f0f0、太字
   - セル: パディング 4pt
   - 列幅:
     - 利用者名: 60%
     - 当月残高: 40%（右寄せ）

3. **ユニット合計**:
   - テーブルの下、右寄せ
   - フォントサイズ: 11pt、太字
   - 上マージン: 8pt

## 🎯 実装イメージ

### レイアウト構造

```
View (unitBox)
├── Text (unitName) - "ユニットA"
├── View (table)
│   ├── View (tableHeader)
│   │   ├── Text (tableHeaderCell) - "利用者名"
│   │   └── Text (tableHeaderCell) - "当月残高"
│   ├── View (tableRow)
│   │   ├── Text (tableCellName) - "山田太郎"
│   │   └── Text (tableCellAmount) - "¥5,000"
│   └── View (tableRow)
│       ├── Text (tableCellName) - "鈴木一郎"
│       └── Text (tableCellAmount) - "¥15,000"
└── View (unitTotalRow)
    ├── Text (unitTotalLabel) - "ユニット合計:"
    └── Text (unitTotalValue) - "¥20,000"
```

## 📝 実装時の考慮事項

1. **テーブルの幅**: 
   - ユニットボックスの幅に合わせる
   - パディングを考慮

2. **利用者が多い場合**: 
   - テーブルが自動的に縦に伸びる
   - ページを跨ぐ場合は次のページに続く

3. **金額の表示**:
   - 右寄せで統一
   - 桁揃えを考慮

4. **ボーダー**:
   - 外側のボーダー（ユニットボックス）
   - テーブルのボーダー
   - 行間のボーダー

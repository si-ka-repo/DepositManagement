## 📐 レイアウト仕様

### 構造
1. **ユニット名**: 
   - 太字、12pt
   - 上マージン: 8pt
   - 下マージン: 8pt

2. **利用者情報エリア（枠組み付き）**:
   - ボーダー: 1px solid #000
   - パディング: 8pt
   - 1行目（利用者名）:
     - 横並びで表示
     - フォントサイズ: 10pt
     - 利用者間の間隔: 20pt
     - 下マージン: 4pt
   - 2行目（当月残高）:
     - 利用者名の下に配置
     - 横並びで表示
     - フォントサイズ: 10pt
     - 右寄せ
     - 利用者間の間隔: 20pt（利用者名と揃える）

3. **ユニット合計**:
   - 枠組みの下、右寄せ
   - フォントサイズ: 11pt、太字
   - 上マージン: 8pt

## 🎯 実装イメージ

### レイアウト構造

```
View (unitBox)
├── Text (unitName) - "ユニットA"
├── View (residentsContainer) - 枠組み
│   ├── View (residentNameRow)
│   │   ├── Text (residentName) - "山田太郎"
│   │   └── Text (residentName) - "鈴木一郎"
│   └── View (residentValueRow)
│       ├── Text (residentValue) - "¥5,000"
│       └── Text (residentValue) - "¥15,000"
└── View (unitTotalRow)
    ├── Text (unitTotalLabel) - "ユニット合計:"
    └── Text (unitTotalValue) - "¥20,000"
```

## 📝 実装時の考慮事項

1. **利用者名と残高の位置合わせ**:
   - 利用者名と残高が縦に揃うように配置
   - 利用者間の間隔を統一（20pt推奨）

2. **利用者が多い場合**: 
   - `flexWrap: "wrap"`で自動的に折り返し
   - または固定幅で複数行に分割

3. **金額の表示**:
   - 右寄せで統一
   - 桁揃えを考慮（固定幅フォントまたは幅指定）

4. **枠組み**:
   - 外側のボーダー（ユニットボックス）
   - 利用者情報エリアのボーダー（内側の枠）
